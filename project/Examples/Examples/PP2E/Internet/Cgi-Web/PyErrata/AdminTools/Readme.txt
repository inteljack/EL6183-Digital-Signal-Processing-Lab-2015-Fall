Tools for errata list maintainer (i.e., product author).
Run dump* to display dbase contents in textual form,
verify* to verify or reject unverified records, and 
backup* to make a gzip tar file dbase backup.  All 3
come in multiple flavors--one for each supported file
medium.  These are all simple command-line interactive
programs; full-blown Tkinter or CGI interfaces, as well 
as converters between flat file and shelve storage 
mediums are left as an exercise.
